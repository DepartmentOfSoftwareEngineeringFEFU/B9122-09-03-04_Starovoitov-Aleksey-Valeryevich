# Система распознавания лиц с интеграцией 1С

## Шаг 1. Развертывание базы данных 1С

1. Перейдите на сайт [https://online.1c.ru/catalog/programs/program/28765768/](https://online.1c.ru/catalog/programs/program/28765768/) и получите бесплатную **«Платформа 1С:Предприятие 8.3, учебная версия (8.3.27.1508)»**.
2. Установите платформу.
3. Запустите **Конфигуратор**.
4. В разделе **Администрирование → Загрузить информационную базу** выберите файл `АвтоматизацияУчетаПосещаемостиУспеваемости.dt` (файл доступен в репозитории).
5. Подтвердите все изменения.

---

## Шаг 2. Создание директории системы распознавания

Выберите место для размещения Python-файлов, изображений и прочих компонентов системы.

> **Рекомендуемый путь:** `C:\dev\Diplom`  
> *При использовании другого пути необходимо изменить содержимое переменной `HOME` в файлах `App.py` и `build_embeddings.py`.*

### Структура папок

Скопируйте содержимое репозитория в выбранную папку или создайте вручную следующую структуру:

```bash
- images_input
- known_faces
- results
- runs
  - detect
    - train
      - weights
- valid_embeddings
- video_input
```

### Наполнение папок

- **`known_faces`** — содержит папки, названные по ФИО студента. Внутри каждой папки — N изображений лица соответствующего студента.
- **`weights`** — скопируйте содержимое папки `weights` из репозитория.
- **`video_input`** — поместите обрабатываемое видео.

### Установка зависимостей

Создайте виртуальное окружение и установите необходимые библиотеки:

```bash
pip install torch
pip install numpy
pip install opencv-python
pip install -U scikit-learn
pip install matplotlib
pip install ultralytics
pip install insightface
pip install onnxruntime-gpu
```

## Шаг 3. Настройка рабочего места в 1С

1. Запустите 1С:Предприятие (вход под пользователем Администратор, пароль отсутствует).
2. В левой панели перейдите в раздел Рабочее место → Путь к Python скрипту.
3. Укажите путь к корневой папке системы распознавания (например, C:\dev\Diplom).
4. Нажмите Записать и закрыть.
